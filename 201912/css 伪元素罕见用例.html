# css 伪元素罕见用例

参考资料：Uncommon Use Cases For Pseudo Elements

1. 标题前放置一个圆，鼠标放上之后圆圈变大

.section-title:before {
    content: "";
    width: 20px;
    height: 20px;
    background: blue;
    /* Other styles */
}

.section-title:hover:before {
    transform: scale(1.2);
}

2. 背景图和渐变背景颜色叠加

假设有一个带有背景图像的元素，并且设计中有一个渐变叠加层，其混合模式设置为color

.bg {
  position: relative;
  height: 300px;
  background: url("image.jpg") center/cover;
}

.bg:after {
  content: "";
  position: absolute;
  left: 0;
  top: 0;
  width: 100%;
  height: 100%;
  background-image: linear-gradient(180deg, #851717 0%, #30328C 100%);
  mix-blend-mode: color;
}

4. 华丽的分割线

在此示例中，存在带有“华丽的分割线”的分隔线。在每一侧都有一条线。使用伪元素和 Flexbox 可以做到这一点

<p>华丽的分割线</p>
1
p {
  display: flex;
  align-items: center;
}

p:before, p:after {
  content: "";
  height: 2px;
  background: #c5c5c5;
  flex-grow: 1;
}

p:before {
  margin-right: 10px;
}

p:after {
  margin-left: 10px;
}
