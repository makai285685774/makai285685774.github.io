# 自定义复选框checkbox实践

<input type='checkbox'> 复选框 默认情况下显示为方形，并在激活状态方形中间出现对勾，并允许取消激活状态，对勾消失

原生 checkbox 的缺点是无法进行设计，这一点在ui框架中已经得到了验证，因为所有ui框架都会自定义复选框，当然这也是本文想要达到的目标，让我们去了解这中间到底经历了什么吧

使用css伪元素和背景图片

<div class="c-checkbox">
    <input class="sr-only" id="c" type="checkbox">
    <label class="c-checkbox__label" for="c">Custom Checkbox</label>
</div>

既然要自定义，原生的<input type='checkbox'>从视觉上应该消失，但是还要保留到dom中，从ui中隐藏的方式如下：

.sr-only {
    position: absolute;
    clip: rect(1px, 1px, 1px, 1px);
    padding: 0;
    border: 0;
    height: 1px;
    width: 1px;
    overflow: hidden;
}

	background: url('./img/checkmark.svg') center/12px no-repeat;
