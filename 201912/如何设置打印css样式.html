# 如何设置打印css样式

@media print { ... } 参考链接

当我们想要打印某一页时，为了打印的效果更加友好，总要做一些特殊的处理 比如

  1. 大的标题不应该出现在页面底部，换行似乎时更好的选择

  2. 避免一个表格跨页打印

  3. 就算是跨页也要加上表格标题

  4. 超链接内容能够打印出链接地址

  5. 特定页面的边距

  6. 打印的一些其他显示或隐藏内容

    等等

接下来我们逐一解决，css写起来都很简单

相信大家都写过媒体查询@media screen(condition){} 来区别不同窗口大小下的css样式，其实@meadia print{}也是可以做到一些特殊的要求，只要在{ }加入css , 下面进入实战环节

1. 避免大标题出现在页面底部

您可以使用page-break-before，page-break-after和 page-break-inside 您可以将这些属性的值设置为always 或avoid

h1 {
/* h1元素始终始于新页面的顶部 */
page-break-before: always;
}

section.city-map {
/* 此部分始终占据其自己的页面. */
page-break-before: always;
page-break-after: always;
}

2. 避免表格跨越多个页面

table {
  /* 表尽可能不跨页拆分. */
  page-break-inside: avoid;
}

3. 跨页表格重复标题

只需在<table>中加入<thead> 和<tbody>即可

<table>
  <thead>
    <tr>
      <th>City</th>
      <th>Population</th>
  </thead>
  <tbody>
    <tr>
      <td>Sydney</td>
      <td>4.627 million (2018)</td>
    </tr>
  </tbody>
</table>

4. 打印的时候显示链接内容

可以使用:after伪元素来做到这一点

@media print {
  a[href]:after {
    content: " (" attr(href) ")";
  }
}

5. 特定页面边距问题

使用该@page规则，您可以自定义特定页面的页面边距

@page:first {
  /* 第一页没有边距. */
  margin: 0;
}

@page {
  /* 其他页边距为2cm. */
  margin: 2cm;
}

6. 显示和隐藏问题

@media print {
  /* 打印时隐藏. */
  nav {
    display: hide;
  }
  /* 打印时显示 */
  .watermark {
    display: initial;
  }
}
