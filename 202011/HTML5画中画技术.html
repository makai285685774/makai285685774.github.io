# HTML5画中画技术

今天闲来无事，无意中翻看张鑫旭博客, 其中介绍谷歌浏览器原生支持视频播放的画中画，觉得挺有意思，记录一下 没赚以后用得着。

所谓的画中画就是，右下角出现一个脱离定位的播放窗口，即使浏览器最小化，此视频依然会在右下角播放

几个常用的API简介

//进入画中画
video.requestPictureInPicture();

//退出画中画
document.exitPictureInPicture();

是不是超级简单，为了监听视频播放状态，还提供两个事件

// 进入画中画模式时候执行
video.addEventListener('enterpictureinpicture', function() {
  // 已进入画中画模式
});

// 退出画中画模式时候执行
video.addEventListener('leavepictureinpicture', function() {
  // 已退出画中画模式
});

最后，还提供了 PictureInPictureWindow对象，也就是画中画窗口对象，包含width，height和onresize这些属性



let pipWindow = await video.requestPictureInPicture();  //返回promise
 //log(' 视频窗体尺寸为：'+ pipWindow.width +' x ' + pipWindow.height);

video.addEventListener('enterpictureinpicture', function(event) {
 
  pipWindow = event.pictureInPictureWindow;
  //log(' 视频窗体尺寸为：'+ pipWindow.width +' x ' + pipWindow.height);
  
  // 改变窗口大小触发
  pipWindow.addEventListener('resize', onPipWindowResize);
});
