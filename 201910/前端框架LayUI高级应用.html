# 前端框架LayUI高级应用

本文的目的是记录一下 layui 里简化开发的功能，方便以后使用，其实官方文档说的已经很详细了，不过找起来需要一定的熟悉过程

# 1. 修改单选按钮的样式

layui的单选按钮放弃了原生丑陋极难美化的radio标签,而是采用<input type='radio'>的方式模拟了一个单选按钮，页面生成元素如下：

<div class="layui-unselect layui-form-radio my">
    <i class="layui-anim layui-icon">图标</i>
    <div>文字</div>
</div>
<style>
    .my .layui-form-radio {
        //这里写覆盖样式
    }
    .my .layui-form-radio i{
        //这里是按钮图标
        display:none;  //因为我的需求不需要此按钮
    }

</style>

# 2. 给单选框赋值为空

一点尝试：

<form class='layui-form' lay-filter='demo'>
...
<input type='radio' name='sex' title='男' value='1'>
<input type='radio' name='sex' title='女' value='0'>
..
<button class='layui-btn' lay-filter='sub'></button>
</form>

form
//js 
//省略部分代码
layui.use(['form']，function(){
	form.val('demo',{
		sex:'' //给sex置空
	})
})

看上去上面的代码没什么问题，都是按照layui的规范，个form赋值，结果是

// 监听提交
form.on('submit(sub)',function(data){
	event.preventDefault(); //阻止跳转刷新
	console.log(data.field) //提交的数据
	// 我们发现{ sex: '0'或 '1'}
})

我们发现sex的value值只能是’0‘或’1‘中的一种，这也似乎合情合理，单选框的作用也就是在name='sex' 组里找到唯一的一个value值，所以赋值为空的操作没有成功

既然只能在name='sex' 组里寻找值，我们不妨加一条<input type='radio' name='sex' value=''> 的占位标签，然后再执行

layui.use(['form']，function(){
	form.val('demo',{
		sex:'' //给sex置空
	})
})
