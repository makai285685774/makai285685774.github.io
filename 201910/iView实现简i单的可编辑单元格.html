# iView 实现简单的可编辑单元格

背景：手头进销存的项目，进行到填写单据时，之前采用的弹窗输入from表单已经不能满足前端设计需求，通过浏览 iView 官网，一种简单明了的方式呈现在了眼前

<template>
  <Divider>
     <Button @click="handleAdd">增加</Button>
  </Divider>
  <Table :columns="columns" :data="data"  @on-row-dblclick='handleEdit' height='500'>
       <template slot-scope="{ row, index }" slot="action">
      <div v-if="editObj.editIndex === index">
        <Button @click="handleSave(index)">保存</Button>
        <Button @click="editObj.editIndex = -1">取消</Button>
      </div>
      <div v-else>
        <span>{{ index + 1 }}</span>
      </div>
    </template>
    <template slot-scope="{ row, index }" slot="name">
      <Input type="text" v-model="editObj.editName" v-if="editObj.editIndex === index" />
      <span v-else>{{ row.name }}</span>
    </template>

    <template slot-scope="{ row, index }" slot="age">
      <Input type="text" v-model="editObj.editAge" v-if="editObj.editIndex === index" />
      <span v-else>{{ row.age }}</span>
    </template>

    <template slot-scope="{ row, index }" slot="birthday">
      <DatePicker type="date" :value='valueToDate(row)' @on-change='handelDateChange' v-if="editObj.editIndex === index" placeholder="Select date" style="width: 200px"></DatePicker>
      <span v-else>{{ getBirthday(row.birthday) }}</span>
    </template>

    <template slot-scope="{ row, index }" slot="address">
      <Input type="text" v-model="editObj.editAddress" v-if="editObj.editIndex === index" />
      <span v-else>{{ row.address }}</span>
    </template>

 
  </Table>
   <Divider>华丽的分割线</Divider>
  {{ editObj }}
  
   <Divider />
  {{ data }}
</template>

js 部分如下：

<script>
  export default {
    data () {
      return {
        columns: [
        
          {
            title: '操作',
            width:100,
            slot: 'action'
          },
          {
            title: '姓名',
            slot: 'name'
          },
          {
            title: '年龄',
            slot: 'age'
          },
          {
            title: '出生日期',
            slot: 'birthday'
          },
          {
            title: '地址',
            slot: 'address'
          }
        ],
        data: [
          {
            name: '王小明',
            age: 18,
            birthday: '919526400000',
            address: '北京市朝阳区芍药居'
          },
          {
            name: '张小刚',
            age: 25,
            birthday: '696096000000',
            address: '北京市海淀区西二旗'
          },
          {
            name: '李小红',
            age: 30,
            birthday: '563472000000',
            address: '上海市浦东新区世纪大道'
          },
          {
            name: '周小伟',
            age: 26,
            birthday: '687024000000',
            address: '深圳市南山区深南大道'
          }
        ],
        editObj:{
          editIndex: -1,  // 当前聚焦的输入框的行数
          editName: '',  // 第一列
          editAge: '',  // 第二列输入框
          editBirthday: '',  // 第三列输入框
          editAddress: '',  // 第四列输入框
        }
   
      }
    },
    methods: {
      handleEdit (row, index) {
        
        this.editObj.editName = row.name;
        this.editObj.editAge = row.age;
        this.editObj.editAddress = row.address;
        this.editObj.editBirthday = row.birthday;
        this.editObj.editIndex = index;
      },
      handleSave (index) {
        this.data[index].name = this.editObj.editName;
        this.data[index].age = this.editObj.editAge;
        this.data[index].birthday = this.editObj.editBirthday;
        this.data[index].address = this.editObj.editAddress;
        this.editObj.editIndex = -1;
      },
      getBirthday (birthday) {
        const date = new Date(parseInt(birthday));
        const year = date.getFullYear();
        const month = date.getMonth() + 1;
        const day = date.getDate();
        return `${year}-${month}-${day}`;
      },
      translateDate(str){
        //str 2019-01-02
        var year=parseInt(str.split('-')[0])
        var month=parseInt(str.split('-')[1])-1
        var day=parseInt(str.split('-')[2])
        
        var date=new Date()
        date.setFullYear(year)
        date.setMonth(month)
        date.setDate(day)
        
        return date
      },
      handleAdd(){
        this.data.push({
          name:'',
          age:'',
          birthday:new Date().getTime().toString(),
          address:''
        })
      },
      valueToDate(row){
        let date=new Date(parseInt(row.birthday))
        return date
      },
      handelDateChange(formDate){
        let date = this.translateDate(formDate)
        this.editObj.editBirthday=new Date(date).getTime().toString()
      }
    }
  }
</script>

重点介绍:

  1. @on-row-dblclick 代表table的行数据双击事件，之所以用双击而不用单击事件，是因为日期进入编辑状态时，还要单击更改日期，这样无形中又一次触发了行单击事件

  2. <DatePicker>本身提供了v-model双向绑定，但是一般后台返回的日期数据类型为字符串或时间戳， 而<DatePicker>只支持 javascript 中的Date类型，所以直接v-model并不可行

  3. 最后一个需要注意的点，日期的格式化和反格式化（视具体情况而定）有待改进

并不完整的地方

这只是我在页面当中调试的一个demo,实际用到的肯定要比这复杂得多，所有还要诸多小的功能并没有实现，比如只有添加行而没有删除行，其实很简单一个splice(index,1)方法就搞定了
