# 基于scroll-better实现移动端下拉和tap的vue组件封装

better-scroll 是一款重点解决移动端（已支持 PC）各种滚动场景需求的插件。它的核心是借鉴的 iscroll 的实现，它的 API 设计基本兼容 iscroll，在 iscroll 的基础上又扩展了一些 feature 以及做了一些性能优化。

介绍：原生js当中并没有tap事件，click存在300ms的延迟，原生的滚动条比较生硬，恰巧better-scroll能同时实现tap和scroll ,所以干脆就封装一个移动端的组件

效果如图：

具体看代码：

<template>
  <div class="wrapper" ref="wrapper">
    <div>
      <div class="loading" v-if="isLoading">
         <!-- 此处替换gif动画,或css3 -->
        <img src="../../assets/pulldown_loading.gif" alt />
        <div class="loading">{{ loadMsg }}</div>
      </div>
      <slot></slot>
    </div>
  </div>
</template>
<script>
import BScroll from "better-scroll";
export default {
  name: "BetterScroll",
  props: {
    refresh: {
      type: Function,
      default() {
        return function() {};
      }
    },
    pulldown:{
        type: Function,
        default() {
            return function() {};
        }
    },
    isLoading:{
        type:Boolean,
        default(){
            return false
        }
     },
     loadMsg:{
        type:String,
        default(){
            return '正在加载...'
        }
     }
  },
  mounted() {
    var scroll = new BScroll(this.$refs.wrapper, {
      tap: true,
      probeType: 1
    });

    scroll.on("scroll", pos => {
      if (pos.y > 20) {
        this.pulldown()
      }
    });
    scroll.on("touchEnd", pos => {
      if (pos.y > 20) {
        
        this.refresh();
      }
    });
  }
};
</script>
</script>
<style scoped>

.wrapper {
  overflow: hidden;
}
.loading {
  text-align: center;
  font-size: 12px;
}
.loading img {
  height: 50px;
}
</style>

如何使用：

  <better-scroll :refresh='refresh' :pulldown='pulldown' :isLoading='isLoading' :loadMsg='loadMsg'>
    <ul>
      <li v-for="item in list" :key="item">{{ item }}</li>
   </ul>
</better-scroll>

将动态数据包在better-scroll内，传入对应函数即可，isLoading为布尔值，用来显示或隐藏加载动画，loadMsg代表加载时的提示文字

	refresh(){
      setTimeout(()=>{  //模拟请求 真实场景中用ajax
        this.loadMsg='加载完成'  //可自定义
        setTimeout(()=>{this.isLoading=false},500)  //延迟为了看到 loadMsg改变的值
        
      },2000)
    },
    pulldown(){
      this.isLoading=true
      this.loadMsg='拼命加载中...'  //可自定义
    }
