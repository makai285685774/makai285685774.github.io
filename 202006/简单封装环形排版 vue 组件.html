# 简单封装环形排版 vue 组件

html容器默认都是矩形的，如果我们想让容器内的文字按照椭圆排版该如何实现呢？

引用： 张鑫旭博客

完整代码：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>环形排版组件</title>
</head>
<style>
    .circle{
        
        background-color: skyblue;
        color: #fff;
        border-radius: 50%;
        font-size: 14px;
        padding: 5px;
        box-sizing: border-box;
    }

    .circle before {
        float: left;
        width: 50%; 
        shape-outside: radial-gradient(farthest-side ellipse at right, transparent 100%, red);
    }
    .circle after {
        float: right;
        width: 50%;
        shape-outside: radial-gradient(farthest-side ellipse at left, transparent 100%, red);
    }
</style>
<body> 

    <div id="app">
        <circle-layout :width="180">
            在CSS Sh局问世之前，文字像杂志一样的任意形状的排版几乎是不可能的一直都是用网格、盒子和直线构造局问世之前，文字像杂志一样的任意形状的排版几乎是不可能的一直都是用网格、盒子和直线构造局问世子和直线构造局问世子和直线构造局问世子和直线构造局问世之前，文字像杂志一样的任意形状的排版几乎是不可能的一直都是用网格、盒子和直线构造apes布局问世之前，文字像杂志一样的任意形状的排版几乎是不可能的一直都是用网格、盒子和直线构造。
        </circle-layout>
    </div>
    
</body>
</html>
<script src="https://cdn.jsdelivr.net/npm/vue@2.6.11"></script>
<script>
    Vue.component('circle-layout',{
        template:`<div class="circle" :style="{width:width + 'px'}">
            <before></before><after></after>
            <slot></slot>
        </div>`,
        props:{
            width:{
                type:Number,
                default:200
            }
        },
        mounted() { //根据内容设计算高度
         let contentH = document.querySelector('.circle').offsetHeight; 
         let reactArea = (this.width + 10)  * contentH  //矩形面积 10为容错处理
         //console.log('矩形面积：' + reactArea)
         //椭圆面积  Math.PI *a * b
         
         let height = (reactArea/(Math.PI * this.width/2)) * 2

         document.querySelector('.circle').style.height = height + 'px'
         document.querySelector('.circle before').style.height = height + 'px'
         document.querySelector('.circle after').style.height = height + 'px'
         //实践中发现 通过矩形和椭圆面积换算高度还是粗略的计算，使用中需要传入width值以确保内容完整展示
        }
    })
    new Vue({ el:'#app'})
</script>

说明：环形排版用到的CSS属性是 shape-outside,CSS shapes教程, 矩形拉伸成椭圆形 高度势必发生变化，更准确的说高度变大，我们这里是利用面积不变的原理 ，改变容器的高度，但是也只是粗略的计算，所以根据实际经验在计算矩形面积时宽度增加了10，确保容器内容能够完整的展示
